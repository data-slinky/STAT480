What affects flight delays and cancellations?

University of Illinois at Urbana-Champaign
STAT 480 - Project Group 3

Data

BTS 97-98 Flights data + BTS Carriers data + OpenFlights airports data + FAA aircraft data

Analysis

As any frequent flier knows, flight delays and cancellations are an inevitable part of commercial air travel.
We want to see if there are any particular trends with flight delays and cancellations using a variety of air
travel data. Using Hive, a data warehouse built atop Hadoop, we pre-process the data to select the relevant
information to feed into R. The analytics is presented visually to better identify patterns and trends. We
partition the analysis into four categories. Click the icons below for an in-depth look.
